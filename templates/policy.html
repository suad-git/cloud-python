Memory Cache Parameters

1. There are more than one choice for the capacity of the memory cache, but the default capacity is 5MB.

2. Replacement Policies:

a. Random Replacement
Its the default replacement policy of memory cache, Randomly selects a key and discards it to make space when necessary. This algorithm does not require keeping any information about the access history.

b. Least Recently Used
Discards the least recently used keys first. This algorithm requires keeping track of what was used when, if one wants to make sure the algorithm always discards the least recently used key.
